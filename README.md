# Claude Code for Termux

> ## This repo is archived. Use the proper package instead:
>
> ### **[gtbuchanan/claude-code-termux](https://github.com/gtbuchanan/claude-code-termux)**
>
> ```bash
> curl -fsSL https://raw.githubusercontent.com/gtbuchanan/claude-code-termux/main/install.sh | bash
> ```
>
> It installs as a real Termux `.deb` with automatic dependency resolution, versioned binaries, TMPDIR handling, download caching, a schedulable updater, clean uninstall, and CI that catches Anthropic release breakage. All the core workarounds (patchelf, the `process.execPath` binary patch, the compiled C wrapper) are his work.

---

The scripts in this repo were an early packaging of those fixes into standalone install scripts. They worked, but the upstream project does everything better. Use that instead.
